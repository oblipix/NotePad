Instruções de uso

Bem-vindo ao NotePad! Este é um aplicativo simples e prático para anotar rapidamente. Aqui você pode armazenar ideias, tarefas ou lembretes de forma eficiente e sem complicações.

Como usar:

  • Digite suas notas: Use a caixa de texto no topo da página para digitar suas notas. O NotePad aceita texto simples.
  • Salvar Localmente: Depois de digitar sua nota, clique no botão "Salvar Localmente" para armazená-la no seu dispositivo.
  • Conectar ao GitHub: Se você quiser armazenar suas notas remotamente, pode conectar o NotePad ao GitHub clicando no botão "Conectar ao GitHub". Isso permitirá que você salve as notas na sua conta do GitHub.
  • Visualizar: Ao lado da caixa de texto, você pode ver uma pré-visualização em tempo real da sua nota. Ideal para visualizar como obter seu markdown antes de enviar para o GitHub.

Sobre o NotePad

NotePad é uma aplicação simples para tomar notas, desenvolvida por Maria Carolina, uma desenvolvedora front-end. O objetivo desta aplicação é oferecer uma plataforma prática e eficiente para anotações rápidas, com a possibilidade de salvar localmente e até conectar ao GitHub para armazenar as notas remotamente.

Essa ferramenta foi criada com a intenção de facilitar o armazenamento de ideias, tarefas e lembretes de forma rápida e intuitiva.
